# Project Cockpit

Local-first starter project for learning Codex by building an adidas price tracker with a scraper and LLM reaction pipeline.

## Structure

- `docs/worklog/`: session logs
- `docs/todos/`: next-step tracking
- `docs/architecture/`: durable design notes
- `memory/`: stable project facts and conventions
- `src/`: minimal static app
- `data/`: scraper inputs, snapshots, offers, and reactions

## Direction

The `v2` goal is to track premium adidas products across retailers, compare scraped prices, and generate LLM outputs from spreads and changes. See `north-star.md` for the current plan.

## Open

Open `index.html` in a browser.
